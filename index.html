Html e CSS

O html é uma linguagem de marcação que nos posibilita construir paginas e aplicações web hoje mais responsavel por construir um efeito semantico e proporcionar um melhor posicionamento do site nos mecanismos de busca.

O css é responsavel por construir o efeito visual e deixar o site mais estiloso e proporcionar acessibiliade. É uma ferramenta escencial na construção de paginas e aplicaçõe webs. Gerar um efeito visula bacana e agradavel melhora a navegabilidade2 é proporciona uma esperiencia mais agradavel para p usuario.

Cliente X Servidor

Tambem conecido como Front-end(cliente) e back-end(servidor) são dois mundos que coexistem um mesmos sistema. Para se cria uma pagina dinamica na web é nessesario qeu se use das duas faces.

O Cliente Fronte-end é o contato direto que o usuario terá com o seu trabalho, construir uma palicação para o lado do cliente e tão conplexo quanto construir APIs e aplicaçoes para o lado do servidor. Tendo que se atentar com o uso de meoria de sua aplicação pos sua aplicaçao sera executada pelo navegador e o mesmo estara sendo executado na maquina do cliente, então otimizer a plicação para que tenha uma boa navegabilidade2 é um ponte excencial. Criar interfaces intuitivas e de facil navegabilidade2 tanbém é um dos desafios.

O Servidor Back-end é no entanto posibilida uma ultilização da web mais profunda o armazenamento de dados, contas de emais, cadatros pessoai, cadastro de produtos. Sem os servidores nao seia posivel todas essas funcionalidades da web como comprar um produto, troca de mensagem, serviçoes de strimer de videos. O papel de armagenar e gerenciar dados é todo processado fora da maquina do clienete o site em que o cliente navega é fornecido pelo servido, dessa forma pode se obirter uma navegação mai segura, e a implicação de um proficional back-end é garantir a segurnaça dos dados, proporcionar o correto armazenamento de e geranciamento de dados e tanbém proporcionar uma boa navegabilidade2

Linguagem De Programação

Existem varias linguagem de programação, as linguagem mais populares no momento são: JavaScript, Php, Java, C#. essas são algumas da linguagem de pprogramação para se trabalhar. Uma peculiaridade do java script é que ele pode funcionar tanto do lado do servidor como do lado do cliente nesse caro usa-se o node.js para executar o javaScript fora do navegador.

A linguagem de Porogrmação posibilita essa comunicação e interatividade do cliente com o site. parte escencial na criação de aplicações seja ela web ou não a linguagem de programação é nos abre mundo de possibilidades, contudo a linugagem não é a responsavel por resolver problemas, o programdor resolve problemas e usa da linguagem de programação como ferramenta para criar tal solução.

Aplicação - Web

Aplicação Web, basicamente são aplicativos que são executados pelo navegador. um site do youTube faceBook são exenplos de aplicação web.

Entretanto nessesariamente para se considerar uma aplicação web o site ou aplicativo deve ser executado pelo navegador. O palicativo do faceBook instalado em um smartPhone deixa de ser uma aplicação web pois a estrutura de desenvolvimento desse aplicativo é um pouco diferente de um que roda direto no navegador.

Node js - V8

O nodeJs foi pensaodo apartir de uma sacada dos desenvolvedores de pegar o codigo do V8, motor que executa o javaScript no navegador, e criar uma maneira de rodar o javaScript fora do navegador, assim posibilitando javaScript rodar do lado do servidor. O motor v8 inserido no navegador google chrome e outros é de codigo aberto facilitado a vida dos desenvolvedores
